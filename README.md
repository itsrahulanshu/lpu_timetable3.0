### Prerequisites
- Node.js 18+ 
- LPU UMS credentials

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/itsrahulanshu/timelpu.git
   cd timelpu
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your credentials
   ```

4. **Start the development server**
   ```bash
   npm start
   ```

5. **Open your browser**
   ```
   http://localhost:3001
   ```
